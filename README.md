Paden.github.io
===============

Technologies used: Yeoman's generator-webapp, Bootstrap, FontAwesome, and Wufoo

##Requisites
* Node >= 0.10.25
* Bower

##Installation

```bash
npm install && bower install
```

##Testing

###Test App
```bash
grunt serve
```

###Test Dist
```bash
grunt serve:dist
```

##Deployment
Pushes dist folder to the master branch
```bash
grunt push_gh
```
